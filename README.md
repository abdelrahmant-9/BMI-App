# 🧮 BMI Calculator App

A clean and modern **Body Mass Index (BMI) Calculator** built using **Flutter & Dart**.  
This application allows users to calculate their BMI in less than a minute with a simple and user-friendly interface.

---

## 🚀 About The Project

This is my first Flutter application.  
The goal of this project was to practice:

- Flutter UI design  
- State management basics  
- Handling user input  
- Navigation between screens  
- Applying clean and modern UI concepts  

The app provides a smooth experience from entering data to viewing personalized BMI results.

---

## ✨ Features

- 👤 Enter user name  
- 🎂 Select birth date  
- 🚻 Choose gender  
- 📏 Adjust height (cm)  
- ⚖️ Adjust weight (kg)  
- 📊 Instant BMI calculation  
- 🟢 Result classification (Underweight / Normal / Overweight / Obese)  
- 🔄 Recalculate option  
- 🎨 Clean and modern UI

## 🧠 How BMI Is Calculated

### 📌 BMI Formula

```text
BMI = weight (kg) / (height (m))²
```

### 📌 Example

```text
Height = 1.70 m
Weight = 65 kg

BMI = 65 / (1.70 × 1.70) = 22.49
```

## 📊 BMI Categories

| BMI Range        | Classification |
|-----------------|---------------|
| Less than 18.5  | Underweight   |
| 18.5 – 24.9     | Normal        |
| 25 – 29.9       | Overweight    |
| 30 and above    | Obese         |
